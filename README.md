<h1 id="setup">Setup</h1>


```ruby
# Clone this repository
$ git clone https://github.com/haripriya79/IICWebsite.git
# Go to the repository folder
$ cd IICWebsite
# Go to Backend
$ cd backend
# start the backend server
$ npm start
# Go to frontend
$ cd frontend
# start the frontend server
$ npm start

```
